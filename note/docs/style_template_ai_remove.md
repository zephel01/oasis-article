# AI臭除去テンプレート（各AI特徴対応版）

このテンプレートは ChatGPT / Claude / Gemini / Grok が生成する文章に
共通・固有のAI臭パターンを網羅的に除去するためのスタイルガイドです。
note_rewriter.py の --template オプションで指定して使います。

## 文体の方針
- 口語調・カジュアルなですます体
- 「〜してみた」「〜ハマった」「〜できた」「〜だった」を積極的に使う
- 一人称は省略か「自分」。「筆者」「私たち」は使わない
- 短い文を多用。一文は60字以内を目安に
- 結論を先に、詳細は後に
- 過剰な丁寧表現・持って回った言い回しは削除

## 全AI共通のAI臭パターン
以下は ChatGPT / Claude / Gemini / Grok すべてで頻出するパターン:
- 「〜と言えるでしょう」→ 断定か感想に
- 「〜が求められます」→ 「〜が必要」に
- 「〜を実現しました」→ 「〜できた」に
- 「本記事では〜ご紹介」→ 削除か「ここでは〜」に
- 「いかがでしたか？」→ 削除
- 「はじめに」「概要」の冒頭 → いきなり本題から
- 「まとめ」の教科書的な締め → 感想・次にやりたいことに
- 「〜について解説します」→ 削除。いきなり書く
- 「徹底解説」「完全ガイド」「〜のすべて」→ シンプルなタイトルに
- 「それでは〜見ていきましょう」→ 削除
- 「〜は非常に重要です」→ 「〜が大事」に
- 「〜を活用することで」→ 「〜を使えば」に
- 「〜を踏まえた上で」→ 「〜を考えると」に
- 「〜することが可能です」→ 「〜できる」に
- 「以上のことから」→ 削除か「つまり」に

## ChatGPT 特有のパターン
- 「〜が重要です」の多用 → 本当に重要な箇所だけに限定
- 「素晴らしい質問ですね」的な前置き → 全削除
- 「もちろん！」「ぜひ！」の過剰な肯定 → 削除
- 「以下にまとめました」→ 削除。そのまま書き始める
- 「具体的には以下の通りです」→ 削除
- 「ステップバイステップで」→ 削除か「順番に」に
- 「お役に立てれば幸いです」→ 削除
- 「参考になれば幸いです」→ 削除
- 「ご不明な点があれば」→ 削除
- 「〜に焦点を当てて」→ 「〜について」に
- 過度なナンバリング（1. 2. 3.）→ 本当に手順の場合のみ残す

## Claude 特有のパターン
- 「〜ですね」の多用 → 必要な箇所だけに限定
- 「〜かもしれません」の多用 → 断定できるなら断定
- 「ただし」「一方で」の過剰な留保 → 本当に必要な場合だけ残す
- 「正直なところ」「率直に言うと」→ 削除。そのまま書く
- 「〜という点は注目に値します」→ 「〜がポイント」に
- 「いくつかの点を指摘できます」→ 削除。そのまま指摘する
- 「〜と考えられます」→ 断定か「〜だと思う」に
- 「重要な点として」→ 削除
- 「申し訳ありませんが」的な前置き → 削除
- 過剰なバランス取り（「Aもあるが、Bもある」の羅列）→ 主張を絞る

## Gemini 特有のパターン
- 「ポイントは3つあります」的なナンバリング予告 → 削除。そのまま書く
- 太字（**）の多用 → 本当に強調したい箇所だけに限定
- 聞かれてない補足情報の追加 → 削除
- 「以下に詳しく説明します」→ 削除
- 「〜は以下のとおりです」→ 削除
- テーブル・表の多用 → 本当に比較が必要な場合のみ残す
- 「ご参考までに」→ 削除
- 「〜の観点から」の多用 → 必要な場合だけ残す
- 「網羅的に」「包括的に」→ 削除
- 「複数のアプローチが考えられます」→ 削除。おすすめだけ書く

## Grok 特有のパターン
- 「ぶっちゃけ」「正直なところ」の砕けすぎ → 文体に合わせて調整
- 皮肉やブラックユーモア → 技術記事では削除
- 過度に断定的な表現 → 根拠がある場合のみ残す
- 「まぁ」「ぶっちゃけて言えば」→ 削除か「実際は」に
- Twitter/X ネタの混入 → 削除
- 「〜なんだけど」の口語すぎる表現 → 文体に合わせる
